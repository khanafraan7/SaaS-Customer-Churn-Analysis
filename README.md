# SaaS Customer Churn Analysis 🚀

> **Tech Stack:** Excel | PostgreSQL | SQL | Power BI

## 📌 Project Overview

This end-to-end Business Intelligence project analyzes customer behavior, revenue performance, churn risk, and support operations within a SaaS business.

Using Excel, PostgreSQL, SQL, and Power BI, the project transforms raw customer, payment, engagement, and support data into actionable business insights through data cleaning, customer segmentation, churn analysis, and interactive dashboarding.

---

## 🎯 Business Objectives

- Analyze revenue performance across subscription plans and countries.
- Identify customer churn patterns and retention risks.
- Segment customers based on revenue contribution.
- Evaluate customer engagement and support burden.
- Generate actionable business recommendations.

## 📂 Dataset Structure

The repository contains both raw and cleaned datasets.

### Raw Data
Original datasets generated from Mockaroo.

### Cleaned Data
Datasets after:

- Removing duplicates
- Handling missing values

- Data validation and quality checks

### Tables

#### Customers

## 🔍 Key Insights

### Revenue

- High Value customers generate 69.4% of total revenue.
- Revenue is highly concentrated among premium customers.

### Churn

- Low Value customers have the highest churn rate (39.4%).
- High Value customers maintain the lowest churn rate (9.0%).

### Support

- Low Value customers generate the highest support burden.
- High Value customers require the fewest support interactions.
